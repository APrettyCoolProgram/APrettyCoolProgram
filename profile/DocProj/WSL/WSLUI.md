# WSLUI

## Ubuntu

1. Install Ubuntu (via WSLUI)
2. Move to B:\WSL\Install\Ubuntu\
3. Export to B:\WSL\Install\Ubuntu\Ubuntu-YYYY-MM-DD.tar
4. Compress to B:\WSL\Install\Ubuntu\Ubuntu-Clean-YYYY-MM-DD.tar.7z
5. `wsl`
6. Created username/password
7. `sudo apt update && sudo apt upgrade`
8. Suspended Ubuntu (via WSLUI)
9. Shutdown WSL (via WSLUI)
10. Export to B:\WSL\Install\Ubuntu\Ubuntu-YYYY-MM-DD.tar
11. Compress to B:\WSL\Install\Ubuntu\Ubuntu-YYYY-MM-DD.tar.7z
12. Clone to B:\WSL\Ubuntu-Deven\ as "Ubuntu-Deven"
13. Export to B:\WSL\Ubuntu-Deven\Ubuntu-Deven-YYYY-MM-DD.tar
14. Compress to B:\WSL\Ubuntu-Deven\Ubuntu-Deven-YYYY-MM-DD.tar.7z
